What is space and time complexity

Time Complexity - The amount of time required for an alogorithm to run, as a function of input size

Space Complexity - The amount of memory required for an alogorithm to run, as a function of input size

How to represent time and space complexity

It is done by using Asymptotic notations

these are mathematical tools to represent this

1. Big-O-Notation - Worst Case Complexity

2. Omega-Notation - Best Case Complexity

3. Theta-Notation - Average Case Complexity
